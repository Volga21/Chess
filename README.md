## Ход игры
Для хода необходимо нажать на фигурую, чтобы убрать выбор с фигуры необходимо нажать повторно

## Сборка
Необходимо на пк иметь библиотеку SDL
```
git clone https://github.com/Volga21/Chess
git clone https://github.com/oxygine/oxygine-framework
cd Chess 
mkdir build
cd build
cmake ..
make
./MyChess
```
